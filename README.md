# Git Bash Command Line:


## Change folder:
cd folder_name

## Change privious folder:
cd ..

## Create folder:
mkdir folder_name

## Remove folder:
rmdir folder_name

## Clear prompt:
clear

## What is in our current folder:
ls

## Create file:
touch file_name.txt

## Delete file:
rm file_name.txt

## Read file:
cat file_name.txt

## Write in file:
echo "I learn about Bangladesh" > file_name.txt

## Add something into file:
echo "I love Bangladesh" >> file_name.txt
